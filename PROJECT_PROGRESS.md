## 已实现功能

GUI：

- 导入图片
- 导入文件夹
- 拖拽导入受支持的图片文件或文件夹
- 从图片列表移除选中文件或清空全部导入文件
- 在棋盘格背景上预览当前图片
- 画布支持滚轮缩放、按钮缩放、适应窗口和右键/中键拖拽平移
- 绘制检测到的 sprite Bounding Box
- 显示 sprite 缩略图
- 切图结果预览支持缩略图大小调节，以及横向滚动/自动换行布局
- 支持从画布或缩略图列表选择切图框
- 支持 Alpha 阈值、最小面积、Padding、邻近区域合并和命名模式参数
- 提供去背景、自动切图、导出和批量处理按钮与菜单项
- 去背景参数支持纯色快速模式、AI rembg 模式、四角自动取色、左上角取色、手动 RGB、容差、羽化和去白边
- 支持点击画布吸取背景色并写入手动 RGB 参数
- 切图预览使用棋盘格缩略图，横向滚动展示，避免透明素材在白底上看不清或被布局裁切
- 支持选中切图框后编辑 X/Y/W/H、按步进移动、按步进缩放和删除当前选区
- 左侧控制面板改为滚动区域，画布和预览区降低硬性最小尺寸，窗口可正常缩放

核心：

- 支持图片格式：PNG、JPG、JPEG、WEBP、BMP
- 图片读取时执行 EXIF 方向修正并转换为 RGBA
- 使用 OpenCV 连通区域分析基于 Alpha 通道自动切图
- 支持 Padding 和最小面积过滤
- 支持可选的邻近区域合并模式
- 提供可选的 `rembg` 去背景封装
- 提供快速纯色背景移除算法，支持柔化边缘 Alpha 和边缘背景色污染清理
- 提供切图框编辑辅助函数，用于夹紧边界、移动、缩放和重新编号
- 提供批量处理流水线

导出：

- 导出处理后的透明 PNG
- 导出单个 sprite PNG
- 导出 JSON 元数据

测试：

- 纯色背景移除测试
- 图片加载测试
- 切图框编辑测试
- 自动切图测试
- PNG 和 JSON 导出测试

## 已知限制

- GUI 目前只通过 offscreen 构造验证，尚未完成完整手动交互测试。
- 纯色快速去背景适合背景颜色稳定的素材，不适合复杂自然背景。
- `rembg` 在第一次实际去背景时可能会下载模型文件。
- JPG/BMP 输入会被转换为完全不透明的 RGBA 图片，因此通常需要先去背景，Alpha 切图才有意义。
- 当前切图框编辑支持数值和按钮微调，尚未实现直接拖拽边框或控制点。
- 尚未实现图集 Packing。
